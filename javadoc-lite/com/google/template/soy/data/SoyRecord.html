com.google.template.soy.data

Interface SoyRecord

  • All Superinterfaces:
    SoyValue, SoyValueProvider
    All Known Subinterfaces:
    SoyDict, SoyEasyDict
    All Known Implementing Classes:
    SoyAbstractCachingRecord, SoyAbstractRecord, SoyMapData

    @ParametersAreNonnullByDefault
public interface SoyRecord
extends SoyValue
    A record containing name-to-value mappings referred to as fields. Each name is a string and each value is a SoyValue (can be unresolved).

    Important: Until this API is more stable and this note is removed, users must not define classes that implement this interface.

Interface SoyValue

  • All Superinterfaces:

    All Known Subinterfaces:

    All Known Implementing Classes:

    @ParametersAreNonnullByDefault
public interface SoyValue
extends SoyValueProvider
    Superinterface of all Soy value interfaces/classes. Replaces the old SoyData.

    Important: Until this API is more stable and this note is removed, users must not define classes that implement this interface.

Interface SoyValueProvider

  • All Known Subinterfaces:
    SoyDict, SoyEasyDict, SoyEasyList, SoyList, SoyMap, SoyRecord, SoyValue
    All Known Implementing Classes:
    com.google.template.soy.data.restricted.CollectionData, SanitizedContent, SoyAbstractCachingRecord, SoyAbstractCachingValueProvider, SoyAbstractList, SoyAbstractMap, SoyAbstractRecord, SoyAbstractValue, SoyData, SoyFutureValueProvider, SoyListData, SoyMapData

    @ParametersAreNonnullByDefault
public interface SoyValueProvider
    A provider of a Soy value.

    This allows for adding providers of late-resolved values (e.g. Futures) to records/maps/lists that are only resolved if the values are actually retrieved. Note that each Soy value object should itself be a provider (of itself).

    Important: Until this API is more stable and this note is removed, users must not define classes that implement this interface.
